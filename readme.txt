Woocommerce Products shows on table with filetring,ordering,pagination,lightbox,add to cart,admin settings everything just with single shortcode.

== Description ==

This is a flexible plugin for all users to show the products in table with search,ordering.It has also on click pop up details.Using shortcode [woo_qotv_code category="pen,bag"] user can show it on the post or pages.From plugin settings user can choose header color,cell color,all font size,cart color and cart icon color. 


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Shortcode [woo_qotv_code] for show products on page or post.
4. Only In Stock Products will show on the List

== Frequently Asked Questions ==

= Where can I find Shortcode for it? =

Use [woo_qotv_code category="bag,pen"] where bag and pen is category name.You can use single and multiple both category.

= What is new on 2.1 version? =

In the latest Version 2.1 you will get ordering,show numbers of products,pagination,settings from admin section where u can choose all color, font sizes and button color also.

= You have any PRO version? =

Yes.We have a pro version where you can see muti product add to cart,filtering,ordering,admin section.

= Where i will get PRO version? =

For get PRO version you have to contact shojibvai@gmail.com or go to http://amp-templates.com.

= Where is the settings option for the plugin? =

You can see it under settings menu.Under settings it is called Quick Order table View Settings.

= Can i put shortcode on page or posts? =

Yes you can.But you need to make sure your post/page editor support shortcode.

= Can i change image size on pop up? =

Yes you can if you know basic php and css.We will add all of these featues in new versions.

= Out Of Stock Products Will Show on the table? =

No.Only Products which are on stock will show here on table.
